# Railway Manager

A web-based railway management simulation game that allows you to design and manage railway networks on an interactive map.

## Phase 1 Features

- **Interactive Map**: Built with Leaflet.js and OpenStreetMap
- **Draw Railway Lines**: Create railway lines by clicking points on the map
- **Place Stops**: Add station stops at specific locations
- **Manage Network**: View all lines and stops in the sidebar, click to navigate
- **Edit & Delete**: Rename lines and stops, or remove them from the network

## Getting Started

1. Open `index.html` in a web browser
2. Click "Draw Railway Line" to start creating a railway line
3. Click on the map to add points, double-click to finish
4. Click "Add Stop" to place station markers
5. Click items in the sidebar to zoom to them on the map
6. Click on lines or stops on the map to edit their names

## Technology Stack

- **Leaflet.js**: Open-source mapping library
- **Leaflet.Draw**: Plugin for drawing and editing features on maps
- **OpenStreetMap**: Free map tile provider
- Vanilla JavaScript (no build tools required)
